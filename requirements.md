# 📦 Requirements for DPN Router Project

> **Everything you need to set up your home DPN router using a Windows PC + a VPN client.**

---

## 🧠 Hardware

* 🖥️ **Old laptop or desktop PC** with Windows 10/11.
* 🔌 **Power supply** (must stay ON 24/7).
* 🌐 **Ethernet LAN port** (at least one, two is better).
* 🔌 **USB-to-Ethernet adapter** (if your PC has only 1 LAN port).
* 📡 **Secondary router** (e.g., Huawei) to broadcast the VPN Wi-Fi.
* 🧵 **2x Ethernet cables** (CAT 6 recommended).

---

## 💾 Software

* ✅ **Outline VPN Client** – [getoutline.org](https://getoutline.org) *(or Clash, Amnezia, WireGuard, etc.)*.
* ✅ **`no_sleep_forever.md` commands** – to disable all sleep/power-saving modes.

---

## 🛠️ Network Setup Summary

* **Main Router** = Local IP *(for Smart Home & local state services)*.
* **Secondary Router** = Foreign IP *(for YouTube, Telegram, Netflix, etc.)*.

---

## ⚠️ Notes

* The server must be kept powered on 24/7.
* The secondary router only works when the VPN client is actively connected on the PC.
* ❌ **Do not let the PC sleep or close the laptop lid** *(apply the `powercfg` commands from the guide!)*.

---

## 🎉 Result

Enjoy uncensored internet on all devices connected to the secondary router 🔓🌍  
Perfect for smart TVs, phones, tablets, and laptops!
